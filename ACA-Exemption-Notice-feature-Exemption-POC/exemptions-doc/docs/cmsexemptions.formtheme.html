(static) stepCompletionHandler(somExpression) → {Boolean}

A stepCompletionHandler function which can execute a panel completion expression in addition to the regular guidebridge validation. a new attribute "panelCompletionExpression" must be added to the panel node in CRX. the result of this expression must be an array of {som:, errorText:} (which is what guideBridge.validate generates). an empty array should be returned if the expression validates without error.
